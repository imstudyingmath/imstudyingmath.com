---
title: 体の標数について
---

教科書の、体の標数のところを読んでました。

## 教科書では

> $\Z$ から体 $K$ への自然な準同型を $\phi$ とするとき、
>
> - $Ker(\phi) = (0)$ なら標数は $0$、
>
> - $p$ が素数で $Ker(\phi) = p\Z$ なら標数は $p$ とする

という定義になってました。


## `mathlib` では

`mathlib`を探してみたら(LLMを使いました)

```lean
class _root_.CharP (R : Type*) [AddMonoidWithOne R] (p : outParam ℕ) : Prop where
  cast_eq_zero_iff (R p) : ∀ x : ℕ, (x : R) = 0 ↔ p ∣ x
```

と定義されているみたいです。

でも、これは教科書の定義とは違う。

$Ker$ も $p\Z$ も出てこない…。

「`mathlib`あるある」かもしれません。


## 教科書沿ったで標数を定義すると

一応、教科書になるべく沿って標数を定義すると(LLMも使いながら)、

```lean
import Mathlib.RingTheory.Ideal.Maps

class CharP (K : Type) [Ring K] (p : ℕ) : Prop where
  kernel_eq_span : RingHom.ker (Int.castRingHom K) = Ideal.span { (p : ℤ) }
```

で、エラー無しになりました。

でも、まだちょっとよく分かっていないかもしれない…。

### 教科書に沿った定義の補足

`RingHom.ker (Int.castRingHom K)`のところは、$Ker(\phi)$ のことみたいです。

`Ideal.span ...`の方は、

- `Ideal.span {(p : ℤ)}`は、$p$の倍数全体の集合(つまり$p\Z$)で、
- `Ideal.span {(0 : ℤ)}`は、$0$で生成されるイデアル、つまり$(0)$

ということになるみたいです。

$(0)$なら、みたいな書き方は、しなくていいんですね。


## 例

$Ker(\phi)$ は、

$Ker(\phi) = {x \in A | \phi(x) = 0B$

で、$\phi$ で $A$ から $B$ の $0$ に移るもののこと。

具体的に標数 $0$ のものを探してみたいと思います。

$\Q$ は体で、\Z から $\Qへの$準同型写像は、
\Zのnを\Qの同じ値に移す写像。

そうすると、\Zの0しか\Qで0にならないので、
Ker(\phi) = (0)ということになる。

次に標数がpのものを探すと、

\Z/3\Z = {0, 1, 2} だと、

\Zから\Z/3\Zへの準同型写像は、3で割ったあまりを得る写像。

そうすると、{0, 3, 6, ...}がKer(\phi)なので、
{0, 3, 6, ...}は3で生成されているので、Ker(\phi)=(3)だから
\Z/3\Zの標数は3ということになる、ということなのか…。
